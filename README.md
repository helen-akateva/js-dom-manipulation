# JavaScript DOM Manipulation

A collection of interactive JavaScript exercises demonstrating fundamental DOM manipulation techniques, including element selection, dynamic content generation, and event handling.

## üìù Description

This project contains five practical tasks that showcase core DOM manipulation skills:

1. **Categories Counter** - Parsing nested lists and counting elements
2. **Image Gallery** - Dynamic image gallery generation from data array
3. **Login Form** - Form handling and validation
4. **Name Input** - Real-time input manipulation
5. **Background Color Changer** - Interactive color generator with dynamic styling

Each task is implemented as a separate HTML page with corresponding JavaScript module, demonstrating clean code organization and modern ES6+ syntax.

## üìã Functionality

### Task 1: Categories List
Counts and displays categories with their elements count using nested list traversal.

### Task 2: Image Gallery
Generates an image gallery dynamically from an array of objects using functional programming approach.

### Task 3: Login Form
Implements form validation and data collection with proper event handling.

### Task 4: Name Input
Creates an interactive greeting that updates in real-time based on user input.

### Task 5: Color Changer
Generates random background colors and displays the hex value on button click.
